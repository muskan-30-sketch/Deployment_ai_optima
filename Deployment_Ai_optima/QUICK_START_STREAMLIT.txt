🚀 GET STARTED IN 3 MINUTES
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Install Requirements
─────────────────────────────────────────────────────────────────────────────

  Open Command Prompt / Terminal and run:

  $ pip install -r requirements.txt

  This installs:
  ✓ streamlit (web app framework)
  ✓ pandas (data processing)
  ✓ xgboost & lightgbm (ML models)
  ✓ plotly (interactive charts)
  ✓ scikit-learn (metrics)


STEP 2: Prepare Your Files
─────────────────────────────────────────────────────────────────────────────

  Make sure these files are in the same folder as streamlit_app.py:

  ✓ clean_dataset_numeric.csv           (your data)
  ✓ xgboost_revenue_model.pkl           (trained model)
  ✓ lightgbm_revenue_model.pkl          (trained model)
  ✓ feature_columns.pkl                 (feature names)
  ✓ streamlit_app.py                    (this app)
  ✓ requirements.txt                    (dependencies)


STEP 3: Run the App
─────────────────────────────────────────────────────────────────────────────

  In Command Prompt / Terminal, run:

  $ streamlit run streamlit_app.py

  You'll see:
  
    You can now view your Streamlit app in your browser.

    Local URL: http://localhost:8501


STEP 4: Open in Browser
─────────────────────────────────────────────────────────────────────────────

  Click the link or open your browser and go to:

  http://localhost:8501

  That's it! The app is now running! 🎉

🔧 COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════════════════════

❌ "ModuleNotFoundError: No module named 'streamlit'"
   → Run: pip install -r requirements.txt

❌ "FileNotFoundError: xgboost_revenue_model.pkl"
   → Make sure model files are in the same folder as streamlit_app.py

❌ "Port 8501 already in use"
   → Use different port: streamlit run streamlit_app.py --server.port 8502

❌ App is slow or freezing
   → Try reducing sample size in optimization settings

❌ CSV upload not working
   → Check CSV format: needs UnitPrice, Discount, Quantity, OrderDate, TotalAmount

💡 USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

EXAMPLE 1: Predict Price for One Product
─────────────────────────────────────────
  1. Go to "Price Prediction" page
  2. Enter: Price=$100, Discount=5%, Quantity=5
  3. Click "Predict Revenue"
  4. See predictions: XGBoost, LightGBM, Ensemble
  5. Compare results

EXAMPLE 2: Optimize Prices for 50 Products
───────────────────────────────────────────
  1. Go to "Price Optimization"
  2. Set: Price Range 0.8-1.3x, Discount 0-30%, Sample=50
  3. Click "Run Optimization"
  4. See revenue lift (should be > 18%)
  5. Download CSV with optimal prices

EXAMPLE 3: Upload Your Own Data
────────────────────────────────
  1. Prepare CSV file with columns:
     - UnitPrice, Discount, Quantity, OrderDate, TotalAmount
  2. Go to "Batch Processing"
  3. Click "Upload CSV File"
  4. Select your file
  5. Click "Process Batch"
  6. Download predictions

EXAMPLE 4: Analyze Performance by Month
────────────────────────────────────────
  1. Go to "Performance Analysis"
  2. Select: Metric=R² Score, Group by=Month
  3. Click "Run Analysis"
  4. See performance for each month
  5. Identify seasonal patterns
